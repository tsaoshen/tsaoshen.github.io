Steven Tsao's Project Page

Here are some of the software projects which I accomplished during the BCIT CST diploma program:

Personal Profile Management System

A simple asp.net application which allows the user to add/update/delete personal profiles with graphical user interface. The application is connected to Microsoft Access database via ODBC which manages all the relevant data.

Forked Server Implementation

Multi-client TCP server developed in C++ which allows the user to enter a list of numbers and then returns the result after performing preorder / inorder / postorder traversal as specified in the configuration file. The server uses interprocess communication to handle multiple clients which the parent process receives the input data from the client and the child process performs the algorithm to be returned to the client.

Time Allocation Dashboard

This is the project completed during the Computer Projects Practicum course with a team for the company Gravit-e Technologies. The team implemented the main page of the dashboard which summarizes the basic information of active clients and projects for the company. This project uses Laravel 5.2 as the primary development environment. My roles of the project include implementing SQL queries for the required tables, integrating user interface developed by another team member, and performing unit tests on the data to ensure the numerical values in the tables are consistent with the values in the other pages of the dashboard.

Recipe Book Android Application

Developing an android app with a team of three for the mobile development course. This application allows the user to create or view a recipe for a meal which is stored in the database. My roles include implementing the SQLite database and testing the correctness of all the buttons and functionalities of the app.

Web-Based Brain Game

I developed a web-based brain game "Escape District 34" with a team using Agile methodology for the Projects course during the second semester which allows the user to improve their pattern recognition skills. My roles of the project involve implementing the visual details of the gameplay screen in JavaScript, adding audio effects, and testing whether each button links to the correct gameplay screen.
